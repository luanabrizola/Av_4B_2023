Versão mobile e desktop

Nossa versão mobile utiliza um design responsivo e tem como características um menu de navegação oculto que é ativado por um ícone de menu, um formulário de cadastro otimizado para telas menores, tendo os elementos de entrada mais compactos e um layout na vertical e tudo isso com um estilo e layout próprio para dispositivos móveis, tornando a experiência de navegar em nosso site mais agradável.

Nossa versão Desktop é projetada para dispositivos de telas maiores e tem como características uma barra de navegação que fornece os links para você pode acessar outras páginas, como nesse caso, você pode ir da página conceitos para o cadastro clicando na barra acima, nosso formulário tem elementos de entrada mais amplos, se adequando para telas maiores.

Flask

O Flask é um framework web em Python e serve para criar aplicativos web. No nosso código, ele é utilizado para definir as rotas, renderizar os templates e interagir com o banco de dados.

Media Queries

Em ambas as versões foi utilizado o media Queries no CSS, que garante a adaptação em diferentes tamanhos de tela.

Dark mode

Nossa página "cadastro" em sua versão Desktop conta com uma botão com a funcionalidade de ativar o modo escuro, sua funcionalidade é dada através do JavaScript.
