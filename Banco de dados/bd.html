Banco de Dados

Nosso banco de dados é relacionado a uma agência de viagens, onde o usuário consegue comprar sua passagem, escolhendo a data de ida e volta, a classe e o destino da mesma. É um banco de dados bem estruturado em questão de Modelo Relacional, contendo, dentro das tabelas, chaves primarias e chaves estrangeiras das quais fazem a relação entre as tabelas, além de garantir as relações entre as tabelas, nossas chaves estrangeiras auxiliam a manter a consistencia dos dados. Uma consideração importante a ser feitar é a implementação de índices para otimizar consultas e a documentação abrangente para guiar o desenvolvimento e a manutenção futuros do sistema.

Tabelas Principais

Usuário

Armazena informações sobre os usuários, como CPF, nome, e-mail e senha

Cidades

Armazena dados sobre cidades, incluindo uma chave estrangeira para o estado ao qual pertencem.

Passagens

Registra dados sobre passagens aéreas, incluindo datas, origens, destinos, duração, companhia aérea, quantidade de pessoas e classe de voo.

Destinos

Armazena detalhes sobre destinos turísticos, com uma chave estrangeira para a cidade associada.
